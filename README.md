# love2d-collects
收集各种有用的第三方工具和资料

[Love2d Editor](https://upizpp.neocities.org/index.html)
> 一款简单的IDE，功能却不简单。

> Love2D Editor提供了一系列的功能使得您可以更加轻松地在Love2D中造轮子。

> 按下F5 / F6即可运行Love2D项目，省去了Love2D启动上的一些步骤，自带导出Windows / Android的功能，使得打包游戏变得更加容易。

> 还有许多功能等待您的探索。
